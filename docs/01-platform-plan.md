## The eight layers of the system

The site is designed as a system of independent, complementary layers. Each layer has a specific function.

Defines how the site looks and feels.

Includes:

- Typography
- Color palette
- Visual rhythm
- Use of whitespace
- Editorial hierarchy
- Aesthetic consistency
